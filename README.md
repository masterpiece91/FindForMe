FindForMe
=========

Python Console App that executes robust directory search with additional perks.

Instructions:
--------------------------------------------------------
* FindForMe.py is the entry-point
* For help type:
	* python FindForMe.py --help

Installing Packages:
------------------------

    pip install -r requirements.txt

Future Features:
---------------------------------------------------------
1. Search statistics
2. Deployment ease with "Freeze"
3. Search caching
